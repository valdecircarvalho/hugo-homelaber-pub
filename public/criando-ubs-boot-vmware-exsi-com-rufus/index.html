Como criar Pendrive com Boot do VMware EXSi utilizando a ferramenta Rufus

01-04-2015 — Written by Valdecir Carvalho — 2 min lendo
#blog vmware  #Boot  #como-instalar  #esxi  #homelab  #homelab vmware  #homelaber  #homelabers  #how-to  #laboratório vmware  #passo-a-passo  #Pendrive  #Rufus  #vblogs  #virtualização  #vmware  #vmware blog  #vmware boot pendrive  #vmware homelab 

Olá Homelaber!

Nesse post, vamos aprender como criar um Pendrive com o boot do VMware ESXi para instalação do seu servidor ESXi utilizando a ferramenta Rufus. https://rufus.akeo.ie/

** **

Fazendo o boot e instalando o ESXi no pendrive, você irá economizar um HD no seu servidor ESXi e ainda terá a faciliade de quando quiser testar outra versão ou ainda outro virtualizador, não precisa gastar tempo reinstalando tudo do zero. É só dar o boot pelo pendrive e pronto.

Primeiramente, vamos fazer o download do Rufus no site https://rufus.akeo.ie/ . A versão atual é a 2.1, mas sempre acesse o site para baixar a ultima versão. Eu particularmente prefiro utilizar a versão “portable”, pois não necessita de instalação, mas isso fica a seu critério.

Inicie o Rufus

Clique no ícone e selecione o ISO do VMware

Cliquem em OK. O seu pendrive será apagado.

**  **

Clique em Iniciar e aguarde

Quando o processo finalizar (são 5 etapas), o pendrive está pronto para ser iniciado no seu servidor.

Esse é um procedimento bastante fácil e tranquilo e pode ser utilizado não somente para criar pendrive de boot do VMware ESXi, mas também do Windows Server, Distros Linux, etc.

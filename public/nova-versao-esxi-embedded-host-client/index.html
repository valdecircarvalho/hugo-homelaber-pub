Nova versão ESXi Embedded Host Client

14-09-2016 — Written by Valdecir Carvalho — 1 min lendo
#Client HTML5  #esxi  #ESXi Embedded Host Client  #HTML5 Client  #vmware  #vmware esxi 

Olá Homelabers!

Em conjunto com a nova versão do vSphere HTML5 Web Client, a VMware lançou na semana passada uma nova versão do ESXi Embedded Host Client (v1.8.1 de 09/09/2016) resolvendo alguns bugs da versão anterior. Essa versão ainda não está completa, mas vem melhorando muito desde de o seu lançamento.

Para quem ainda não conhece, o ESXi Embedded Host Client é um client nativo em HTML e JavaScript que substitui o client C+ (vSphere Client) no gerenciamento de hosts ESXi (sem vSphere). É uma alternativa muito bacana em relação ao client Windows no gerenciamento de hosts ESXi sem vCenter ou com a versão free do ESXi . E digo mais: Podem ir se acostumando a utilizar a versão web, pois aposto 1 real, que a próxima versão do ESXi (6.5??) que deverá anunciada na VMworld de Barcelona será gerenciada quase que totalmente por um client web.

Até a próxima!

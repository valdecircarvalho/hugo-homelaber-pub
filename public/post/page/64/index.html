O Veeam Endpoint Backup FREE oferece uma solução simples e grátis para fazer backup dos desktops e laptops Windows. Para quem não conhece, a Veeam é a empresa lider no segmento de soluções de backup para plataformas de virtualização e seus principais produtos são as ferramentas Veeam Availability Suite e Veeam Backup & Replication. Pretendo falar mais profundamente sobre essas ferramentas aqui no blog. Olá HomeLabers! Esse é o segundo post da série VMware HomeLab e iremos falar sobre os preparativos para a instalação do nosso HomeLab. O primeiro passo é verificar e garantir que o hardware onde será instalado o VMware ESXi atender os requisitos mínimos para a instalação. Existe um KB (2017269) da VMware indicando esses requisitos e pode ser consultado aqui. Em resumo necessitamos de:

  • CPU x86 64 bits e 2 cores * 2GB Memória RAM * 1 Placa de rede Após garantir que o seu hardware é suportado para a instalação, o próximo passo é fazer o download do VMware ESXi através do site da VMware.

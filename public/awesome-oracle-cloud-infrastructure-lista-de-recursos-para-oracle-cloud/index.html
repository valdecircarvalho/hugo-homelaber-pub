Awesome Oracle Cloud Infrastructure - Lista de recursos para Oracle Cloud

Olá Homelabers!

Quem me acompanha aqui no blog, sabe que sou fã de listas Awesome e como estou me envolvendo a cada dia mais com Oracle Cloud, resolvi criar uma lista Awesome no GitHub para juntar e colecionar links úteis sobre OCI.

Eu quero que a lista seja um ponto central com novidades, documentação, informações úteis, treinamentos, blogs, twitters, grupos, etc.

Espero muito que esse projeto decole e ajuda a comunidade. Se você tem alguma contribuição ou correção, basta deixar um comentário aqui ou me mande um twitter ou ainda crie um issue no repositório no GitHub.

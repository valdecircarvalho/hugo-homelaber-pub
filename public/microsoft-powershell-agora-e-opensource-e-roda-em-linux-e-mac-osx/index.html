Microsoft Powershell agora é OpenSource e roda em Linux e Mac OSx

18-08-2016 — Written by Valdecir Carvalho — 1 min lendo
#microsoft  #powershell  #powershell github  #powershell no linux  #powershell no mac  #powershell opensource 

Olá Homelabers!

Nota rapidinha! Hoje, 18/08/2016 a Microsoft anunciou que está abrindo o código do Powershell que agora está disponível para Linux e Mac OSx.

É, quando o Satya Nadella disse: “Microsoft loves Linux” ele realmente não estava brincando.

Ainda não consegui testar essa “maravilha”, mas é um novo mundo que se abre. Eu sou fã do Powershell, infelizmente não sei usar o tanto quanto eu gostaria. Mas a novidade é muito bem vinda!

Ao menos para isso, acabou aquela do sysadmin que tem um Mac ser obrigado a criar uma VM Windows para rodar algum script em Powershell para administrar seus servidores Windows por exemplo.

Assim que eu conseguir um tempo para testar o Powershell no Linux e no Mac OSx eu volto aqui para contar a vocês. E pegando o gancho, uma das primeiras coisas que eu quero testar é utilizar o VMware PowerCLI rodando no Mac :)

Seguem alguns links (em Inglês) sobre o anúncio e também o link do Github onde você poderá fazer o download do Powershell para Windows, Linux (.deb e .rpm) e Max OSx.

É isso ai pessoal! Até a próxima!
